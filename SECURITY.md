# Security Policy

Security is critical to DataFrey. We handle database credentials and authentication tokens — we treat every vulnerability report with the highest priority.

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email **[slava+security@datafrey.ai](mailto:slava+security@datafrey.ai)** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond as quickly as possible and work with you to resolve the issue before any public disclosure.

## Scope

This policy covers all packages in this repository: `datafrey`, `datafrey-api`, and `datafrey-mcp`.

## Recognition

We credit reporters in release notes for confirmed vulnerabilities, unless you prefer to remain anonymous.
